Every digital workspace and personal portfolio needs a personal touch, and I recently decided it was time to build my own visual assets. I needed a cohesive set of custom SVG icons to link to my professional platforms, which served as my official introduction to using Inkscape. Jumping into vector graphics and mathematically drawn shapes was a completely new paradigm for me, but it was an absolute blast to learn the new software. It provided the perfect sandbox to show off some creative liberty while building something uniquely tailored to my online presence.

## The Design Philosophy

Because this was my very first experience with Inkscape, my goal was to figure out how to combine simple geometric shapes to build complex, scalable objects. I wanted to design a set of functional icons that represented my professional identity and linked directly to my work.

Here is a breakdown of the core icons I developed:

- **GitHub (The Code Screen):** To represent my GitHub profile and code repositories, I designed a monitor icon featuring a classic code syntax bracket `</>`. I wanted a clean, stylized representation of my coding environment, utilizing a dark, rounded chassis with soft purple and white terminal accents.

- **LinkedIn (The Briefcase):** To bring a touch of traditional, grounded professionalism to my networking links, I created a classic brown briefcase with a central gold clasp. It serves as the perfect visual anchor for my LinkedIn profile.

- **Projects (The Folder):** For showcasing my development work and repositories, I built a projects icon designed as a thick, vibrant yellow folder, personalized with the hand-drawn letters "MDG."

- **Resume (The Document):** Finally, I needed a dedicated resume icon. I designed a stylized document stamped with a bold "MG" at the top, complete with abstract, zig-zagging text lines and bullet points to represent my professional experience and skills.


## The Logo and Final Composition

I also created my own personal logo: a large, glowing purple "MG" monogram. The stylized, overlapping letters feature a vibrant, neon-like drop shadow that pops instantly. This project was the ultimate crash course. Not only did I walk away with a highly personalized logo and a set of crisp, infinitely scalable SVG icons for my portfolio, but it also laid the groundwork for my future design projects by giving me a solid foundational understanding of Inkscape.
